Configure the Brightnode CLI with your API credentials and API URL to enable programmatic access to your Brightnode resources.

<RequestExample>
```sh Command
brightnodectl config [flags]
```
</RequestExample>

## Example

Configure the CLI with your API key:

```sh
brightnodectl config \
  --apiKey "rpaPOIUYYULKDSALVIUT3Q2ZRKZ98IUYTSK2OQQ2CWQxkd01"
```

## Flags

<ResponseField name="--apiKey" type="string">
Your Brightnode API key, which authenticates the CLI to access your account. You can generate an API key from the [Brightnode console](https://www.brightnode.cloud/console/user/settings).
</ResponseField>

<ResponseField name="--apiUrl" type="string" default="https://api.brightnode.cloud/graphql">
The Brightnode API endpoint URL. The default value should work for most users.
</ResponseField>

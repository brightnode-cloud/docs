List all your Nodes or retrieve details about a specific Node by its ID.

<RequestExample>
```sh Command
brightnodectl get node <nodeId> [flags]
```
</RequestExample>

## Example

List all your Nodes with complete field information:

```sh
brightnodectl get node --allfields
```

## Arguments

<ResponseField name="<node_id>" type="string">
The ID of a specific Node to retrieve. If no ID is provided, all Nodes will be listed.
</ResponseField>

## Flags

<ResponseField name="-a, --allfields">
Include all available fields in the output, providing complete Node information.
</ResponseField>

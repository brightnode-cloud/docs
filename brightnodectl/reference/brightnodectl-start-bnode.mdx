Start a stopped Bnode, resuming compute and billing. Use this to restart Bnodes that were previously stopped.

<RequestExample>
```sh Command
brightnodectl start bnode <bnodeId> [flags]
```
</RequestExample>

## Example

Start a stopped Bnode with a custom bid price for spot instances:

```sh
brightnodectl start bnode abc123xyz456 --bid 0.50
```

## Arguments

<ResponseField name="<bnodeId>" type="string" required>
The ID of the Bnode to start. You can find Bnode IDs using the `brightnodectl get bnode` command.
</ResponseField>

## Flags

<ResponseField name="--bid" type="float">
The bid price per GPU in dollars per hour for spot instance pricing. This only applies to Community Cloud Bnodes.
</ResponseField>

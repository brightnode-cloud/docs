Permanently delete a Pod and all its associated data. This action cannot be undone.

<RequestExample>
```sh Command
brightnodectl remove pod <podId>
```
</RequestExample>

## Example

Terminate a Pod by its ID.

```sh
brightnodectl remove pod abc123xyz456
```

## Arguments

<ResponseField name="<podId>" type="string" required>
The ID of the Pod to terminate. You can find Pod IDs using the `brightnodectl get pod` command.
</ResponseField>

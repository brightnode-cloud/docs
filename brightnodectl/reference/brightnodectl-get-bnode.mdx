List all your Bnodes or retrieve details about a specific Bnode by its ID.

<RequestExample>
```sh Command
brightnodectl get bnode <bnodeId> [flags]
```
</RequestExample>

## Example

List all your Bnodes with complete field information:

```sh
brightnodectl get bnode --allfields
```

## Arguments

<ResponseField name="<bnode_id>" type="string">
The ID of a specific Bnode to retrieve. If no ID is provided, all Bnodes will be listed.
</ResponseField>

## Flags

<ResponseField name="-a, --allfields">
Include all available fields in the output, providing complete Bnode information.
</ResponseField>

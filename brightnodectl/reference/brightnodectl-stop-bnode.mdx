Stop a running Bnode to pause compute operations. The Bnode's persistent storage will be retained, but you'll continue to be charged for storage until the Bnode is removed.

<RequestExample>
```sh Command
brightnodectl stop bnode <bnodeId> [flags]
```
</RequestExample>

## Example

Stop a running Bnode.

```sh
brightnodectl stop bnode abc123xyz456
```

## Arguments

<ResponseField name="<bnodeId>" type="string" required>
The ID of the Bnode to stop. You can find Bnode IDs using the `brightnodectl get bnode` command.
</ResponseField>
